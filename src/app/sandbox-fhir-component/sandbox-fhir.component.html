• FHIR Server

    This is an implementation of the FHIR standard using the US Core Implementation guide. We expect client developers to use the server as part of their development activities to work with different data sets.

  • FHIR Query Tool

    FHIR Query Tool can be used to test a FHIR Server using various search parameter combinations. The tool makes it easy for users to query FHIR servers without knowing the specific authorization workflows and FHIR syntaxes for searches.

  • FHIR Conformance Test Tool

    The FHIR Conformance Test Tool provides an easy way for servers to test their conformance to an implementation guide using test data and test scripts.

  • FHIR Bulk Data Client/Server

    The FHIR Bulk Data Client and Server are an implementation of FHIR Bulk Data Proposal outlined at https://github.com/smart-on-fhir/fhir-bulk-data-docs. The client allows one to query a FHIR Server that supports FHIR Bulk Data APIs. The server is an implementation of FHIR using HAPI and it additionally supports the FHIR Bulk Data APIs.

  • SITE Inferno Testing Tool

    The SITE Inferno Testing Tool provides the capability to test FHIR Servers for their conformance to various implementation guides and SMART on FHIR authorization specifications. The tool has been built using the MITRE FHIR Crucible project.
